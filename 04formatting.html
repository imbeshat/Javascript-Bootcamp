strong tag is used with the text contents which has strong importance, seriousness, or urgency.
Browser typically treats these contents as important and renders the content in bold type.
Where as b tag is used to draw attention to the text without indicating that it's more important.

The em tag is used for words or texts that have a stressed emphasis compared to surrounding text.
These texts are typically displayed in italics.
While i tag is generally used to display the technical term, phrase, the important word in a different language. Example - This text is in italics.

The small tag in HTML is used to display the text in small size such as copyright and legal text as compared to other texts. Example - This is a copyright.

The sub tag is used when we want to display a part of text below the baseline as subscript for some typographical reasons. Example - Chemical formula of water is H2O

he sup tag is used when we want to display a part of text as superscript for some typographical reasons. Example - (a + b)2 = a 2 + b 2 + 2ab

The ins HTML tag is used when a range of text is added to a document. Example - This important sentence is added to the document.

The del tag is used to display the texts that has been deleted from the document. Example - The word this is deleted from the document.

The mark tag is used to mark or highlight some text for reference or notation purpose. Example - The text mark is important to the user in this para.
